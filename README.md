# 🌦 Weather Data Pipeline

## 📌 Overview
This project collects real-time weather data using API, stores it in MySQL, and performs analysis.

## 🛠 Tools Used
- Python
- MySQL
- API (Open-Meteo)

## 🔄 Pipeline Flow
API → Python → MySQL → SQL Analysis

## 📊 Features
- Real-time data collection
- Timestamp tracking
- Data analysis using SQL

## 🚀 Result
Built a working data pipeline similar to real-world systems.
